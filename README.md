# 국회의원 미디어 가시성과 재선 가능성 연구

> 제21대 지역구 국회의원을 대상으로 미디어 노출의 양과 질이 제22대 총선 재선 여부에 미치는 영향을 분석한 혼합연구방법(Mixed Methods) 프로젝트

---

# 1. 프로젝트 개요

## 연구 질문

국회의원이 언론에 많이 노출될수록 재선 가능성이 높아질까?

혹은,

부정적 논란과 결합된 미디어 노출은 오히려 재선 가능성을 낮출까?

---

## 핵심 문제의식

기존 연구들은 주로 **기사 수 자체**를 활용해 미디어 효과를 측정하였다.

- 언론에 많이 등장하면 인지도가 높아진다.
- 인지도가 높아지면 선거에 유리하다.

하지만 현실에서는 다음과 같은 반례가 존재한다.

- 논란 때문에 기사 수가 급증할 수 있다.
- 높은 미디어 노출이 오히려 정치적 부담이 될 수 있다.

따라서 본 연구는 단순한 기사 수가 아니라,

> **전체 기사 중 부정적 쟁점 기사 비율**

이라는 개념을 도입하여 미디어 노출의 질적 차원을 측정하였다.

---

# 2. 연구의 핵심 아이디어

## 기존 관점

미디어 노출 ↑ → 인지도 ↑ → 재선 가능성 ↑

## 본 연구의 관점

미디어 노출은 중립적이다.

- 긍정적 이슈와 결합되면 정치적 자산
- 부정적 쟁점과 결합되면 정치적 리스크

즉,

> 중요한 것은 얼마나 많이 보도되었는가가 아니라,
> 어떤 이슈와 함께 보도되었는가이다.

---

# 3. 연구 가설

### H1. 총 미디어 노출량은 재선 가능성에 긍정적 영향을 미친다.

### H2. 부정적 쟁점 기사 비율은 재선 가능성에 부정적 영향을 미친다.

### H3. 부정적 쟁점 기사 비율은 단순 기사 수보다 더 큰 설명력을 가진다.

### H4. 총 기사 수의 효과는 비선형적일 수 있다.

# 10. 분석 모형

## Model 1. 통제변수만

```text
재선여부 ~ 득표격차 + 여당 + 수도권 + 나이 + 성별 + 선수 + 당적변경
```

## Model 2. 총 미디어 노출

```text
Model 1 + 로그_총기사수
```

## Model 3A. 부정 기사 수

```text
Model 2 + 로그_부정적쟁점기사수
```

## Model 3B. 부정 기사 비율 (주모형)

```text
Model 2 + 부정적쟁점비율
```

## Model 4. 비선형 효과

```text
Model 3B + centered(log_total)^2
```

# 12. 핵심 해석

### 1. 총 기사 수는 유의하지 않음

단순히 많이 보도되는 것만으로는 재선 가능성이 높아지지 않았다.

### 2. 부정적 쟁점 비율은 중요한 설명 변수

전체 기사 중 부정적 쟁점 보도의 비중이 높을수록 재선 가능성이 낮아지는 경향이 나타났다.

### 3. 미디어 효과는 양보다 질이 중요

기사 수 자체보다 보도의 내용이 정치적 결과에 더 중요했다.

본 연구는 미디어 노출을 단순한 인지도 지표가 아니라,

# 16. 사례 연구

## 김남국

- 코인 논란
- 대규모 부정 보도
- 총선 불출마

→ 부정적 쟁점 노출이 직접적 정치적 부담으로 작용한 사례

## 이재명

- 사법 리스크 관련 지속적 보도
- 높은 부정 기사에도 당선

→ 정치적 자원이 부정적 효과를 상쇄한 사례

## 박용진

- 높은 인지도
- 당내 계파 갈등
- 공천 탈락

→ 정당 내부 정치가 미디어 효과를 매개한 사례

---

# 17. 사례연구의 함의

- 부정 기사 효과는 일률적이지 않다.
- 정치적 자원과 정당 구조가 중요하다.
- 미디어 효과는 정치적 맥락에 따라 달라진다.
